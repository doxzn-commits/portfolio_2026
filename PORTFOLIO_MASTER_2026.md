## 2. [대표 프로젝트] AI 콘텐츠 자동 발행 시스템 — 클릭 20배, 전환 0→4건

**카피아 · 내부 게시판(뉴스·차량 꿀팁) 콘텐츠 그로스**

### 문제
내부 게시판은 담당자가 콘텐츠를 수기로 작성하던 채널로, 6개월간 CTA 클릭 90건·전환 12건에 그칠 정도로 성과가 정체되어 있었다. 콘텐츠별 품질 편차가 크고 발행량 확대가 구조적으로 불가능했다.

### 가설
"색인되는 글에는 공통 패턴이 있다. 이 패턴을 규칙화해 AI 생성 파이프라인에 강제하면, 발행량과 품질을 동시에 확보해 검색·AI 검색 유입과 전환을 만들 수 있다."

### 실행
- **원고 자동 생성 파이프라인**: CLI에 키워드를 입력하면 완성 HTML이 출력되는 구조 설계 (뉴스·꿀팁 2개 유형)
- **GEO 표준 템플릿**: 정답 박스, 질문형 H2, 정의형 직답, FAQ, 자사 1차 비교표, body 내부 JSON-LD(BlogPosting·FAQPage·BreadcrumbList)로 Google 및 AI 검색(AI Overview, Perplexity, ChatGPT) 인용 가능성 극대화
- **색인 규칙의 프롬프트화**: 색인 성공/실패 글을 비교 분석해 공통 패턴(제목 훅 — 숫자·순위·고유명사·연도 필수, 도입 하드데이터, 시점 고정, 고유 앵글)을 생성 프롬프트 규칙으로 체계화
- **개발 리소스 0 발행**: UI 재구축 없이 기존 raw HTML 에디터를 활용해 즉시 운영 시작
- **전환 중심 설계**: 모든 콘텐츠가 상담 CTA로 연결되도록 템플릿 단계에서 강제, 조회수가 아닌 CTA 클릭·전환으로 측정

### 성과 (2026년 6월, 동월·동일 채널 비교)

| 구분 | 수기 작성 (도입 전) | AI 자동 발행 (도입 후) |
|---|---|---|
| CTA 클릭 | 16건 | **329건 (약 20배)** |
| 상담 전환 | 0건 | **4건 (전환율 1.2%)** |

- 이전 6개월 누적 클릭(90건)의 3.6배를 1개월 만에 달성
- 일별 클릭이 월초 1~4건 → 월말 20~30건으로 우상향, 에버그린 콘텐츠 누적에 따른 복리 성장 패턴 확인

## 6. CRM 마케팅 — 앱푸시 YoY 매출 236% 증대

**핌아시아 (커머스) · 앱푸시/SMS/Email 채널 운영**

### 문제
앱푸시·SMS·Email 전 채널에서 타겟팅과 테스트 체계가 부재했고, 알림 해제율 증가·도달률 저하와 함께 매출이 감소하고 있었다.

### 실행
- **카피 구조화**: 일반적인 기획전 안내 문구를 카테고리별/상품별 유형 구조로 재설계, 상품 직접 노출·가격 USP 극대화로 전환. CTR A/B 테스트로 검증
- **타겟팅 모수 재설계**: "구매 경험 N회 이상 회원은 전환이 용이할 것"이라는 가설로 구매 이력 기반 3개 그룹 신설
- **분기별 고도화**: 1Q 타겟 그룹 개설 → 2Q 기획전별·후킹성 카피 A/B 테스트 → 3Q 클릭률 데이터 기반 메일링 디자인 폼 리뉴얼(배치 순서·디자인)

### 성과
- 앱푸시 카피 구조화로 **YoY 매출액 236% 증대**
- 타겟팅·카피·폼 개선 누적으로 **1Q 대비 4Q 매출 130% 성장**
