## 1. 프로젝트 개요

블로그 자동화 프로젝트는 카피아의 자동차 콘텐츠 운영을 자동화하기 위한 Python 기반 마케팅 자동화 묶음이다.

핵심은 월별 수입차 프로모션 엑셀 파일을 입력하면 네이버 블로그용 HTML, 섹션별 PNG 이미지, 정리 엑셀, 네이버 블로그 임시저장 업로드까지 자동으로 진행하는 프로모션 포스팅 자동화다.

추가로 블로그 본문 추출기, 카테크 콘텐츠 수집기, Google Sheets 기반 과거 블로그 업로드 자동화 스크립트도 포함되어 있어 콘텐츠 제작·수집·정리·게시의 반복 업무를 줄이는 운영 자동화 체계로 볼 수 있다.

---

## 2. 해결한 업무 문제

월별 수입차 프로모션 콘텐츠는 반복성이 높지만 수작업으로 처리하면 시간이 많이 든다.

주요 문제:
- 브랜드별 엑셀 파일을 열어 모델, 연식, 차량가격, 자사금융, 현금 조건을 직접 정리해야 함
- 블로그용 표와 이미지 디자인을 매번 새로 만들어야 함
- 네이버 블로그에 인사말, 인용구, 모델별 이미지, 설명문, 표, 해시태그를 반복 입력해야 함
- 브랜드별 컬럼명이 달라 데이터 누락이 발생할 수 있음
- 업로드 이력이 관리되지 않으면 같은 파일을 중복 처리할 수 있음

이 프로젝트는 엑셀 파싱부터 네이버 임시저장까지 하나의 명령어 흐름으로 연결해 월간 콘텐츠 제작 시간을 크게 절감하도록 설계되었다.

## 3. 프로모션 자동화 핵심 흐름

실행 파일:
프로모션 자동화/naverblog_promotion.py

주요 실행 명령:
- python naverblog_promotion.py --file 2026_04_아우디.xlsx
- python naverblog_promotion.py --file 2026_04_아우디.xlsx --no-upload
- python naverblog_promotion.py --file 2026_04_아우디.xlsx --upload-only
- python naverblog_promotion.py --all
- python naverblog_promotion.py --save-session
- python naverblog_promotion.py --file 2026_04_BMW.xlsx --no-upload --force

처리 흐름:
1. raw 폴더의 엑셀 파일을 읽는다.
2. 파일명에서 연도, 월, 브랜드를 추출한다.
3. 브랜드 설정이 없으면 엑셀 데이터 기반으로 브랜드 config를 자동 생성한다.
4. 엑셀 컬럼을 유연하게 감지해 모델별 프로모션 데이터를 파싱한다.
5. HTML 본문과 캡처용 HTML을 생성한다.
6. 정리 엑셀을 생성한다.
7. Playwright로 HTML을 렌더링한다.
8. data-section 기준으로 섹션별 PNG를 생성한다.
9. Playwright로 네이버 블로그 글쓰기 화면에 접속한다.
10. 제목, 고정 인사말, 인용구, 이미지, section-comment, 실제 네이버 표, 해시태그를 입력한다.
11. 글은 발행하지 않고 임시저장한다.
12. status.json에 DONE 상태를 기록해 중복 처리를 방지한다.

## 4. 엑셀 파싱과 브랜드 자동화

core/excel_reader.py는 openpyxl 기반으로 엑셀을 읽는다.

기본 처리 항목:
- 연식
- 모델명
- 차량가격
- 자사금융 조건
- 현금/타사조건
- 프로모션 없음 모델 분리
- 모델명 줄바꿈 정리
- 브랜드별 섹션 분류
- 다중 시트 파일 처리

BMW처럼 컬럼명이 다른 파일을 처리하기 위해 컬럼명 고정 방식이 아니라 키워드 기반 감지 구조를 사용한다.

예:
- 자사금융: 자사금융, 자사할인
- 현금/타사: 현금, 타사
- 차량가격: 차량가격, 가격
- 모델명: 모델명
- 연식: 연식

core/config/brands.py에는 브랜드별 색상, 해시태그, URL, 섹션 그룹 규칙이 저장된다. 미등록 브랜드는 엑셀 데이터 기반으로 자동 config를 생성하고 brands.py에 저장할 수 있도록 구성되어 있다.

## 5. HTML·이미지 자동 생성

core/html_generator.py는 블로그용 HTML, 캡처용 HTML, 요약 엑셀을 생성한다.

주요 특징:
- 브랜드별 색상을 반영한 HTML 디자인
- Pretendard 폰트 로컬 임베딩
- 2000px 폭 기준 모바일 가독성 고려
- h1, h2, h3 구조 반영
- 모델별 프로모션 표 생성
- section-comment 설명문 생성
- CTA 영역 생성
- 캡처용 HTML에서는 설명문을 숨기는 방식 지원
- 정리 엑셀 생성

core/image_generator.py는 Playwright와 PIL을 이용해 HTML을 섹션별 이미지로 캡처한다.

이미지 생성 방식:
- 전체 페이지를 한 번 캡처한다.
- HTML의 data-section 기준으로 각 영역의 위치를 계산한다.
- PIL로 개별 섹션을 crop한다.
- images/00_header.png, images/01_intro.png, images/02_*.png 형태로 저장한다.

이 방식은 네이버 블로그 모바일에서 표가 작게 보이는 문제를 줄이기 위해 이미지 자체를 큰 폰트와 고정 폭 디자인으로 생성한다는 점이 핵심이다.

## 6. 네이버 블로그 업로드 자동화

core/naver_uploader.py는 Playwright 기반으로 네이버 블로그 글쓰기 화면을 제어한다.

주요 기능:
- 로그인 세션 저장 및 복원
- 글쓰기 화면 진입
- 제목 입력
- 본문 포커스 제어
- 인용구 삽입
- 이미지 업로드
- 실제 네이버 SmartEditor 표 삽입
- 임시저장 클릭
- 도움말/이어쓰기 팝업 처리

네이버 본문 구조:
- 고정 인사말
- 인용구2 인트로 요약
- 검색 안내 텍스트
- 섹션 루프: 인용구2 브랜드 모델라인 프로모션 -> 이미지 -> 설명문
- 인용구2 럭키 프로모션 정보 -> 본문
- 인용구2 프로모션 쉽게 검색하기 -> 실제 네이버 표
- 마무리 문구
- 해시태그
- 임시저장

이미지 삽입 규칙:
- 00_header.png 제외
- 01_intro.png 제외
- 02_*.png부터 삽입
- CTA 이미지는 현재 업로드 제외

표 삽입은 단순 텍스트 붙여넣기가 아니라 SmartEditor의 실제 표 기능을 조작한다. 표 데이터가 인용구 안에 섞이지 않도록 본문 추가 버튼, 본문 포커스 복귀, 대기 로직을 두었다.

## 9. 포트폴리오용 문장

한 줄 소개:
월별 수입차 프로모션 엑셀을 네이버 블로그 임시저장 글까지 자동 변환하는 Python·Playwright 기반 콘텐츠 운영 자동화 시스템을 구축했다.

상세 문장:
수입차 프로모션 콘텐츠 제작에 필요한 엑셀 정리, HTML 디자인, 섹션별 이미지 생성, 네이버 블로그 업로드 과정을 Python 기반으로 자동화했다. 브랜드별 엑셀 파일을 입력하면 모델·연식·가격·자사금융·현금 조건을 자동 파싱하고, 브랜드 색상과 모바일 가독성을 반영한 HTML 및 PNG 이미지를 생성하며, Playwright로 네이버 SmartEditor에 인용구·이미지·실제 표·해시태그를 입력해 임시저장까지 완료하도록 구현했다. 이를 통해 매월 반복되던 블로그 제작·정리·업로드 업무 시간을 크게 절감하고, 콘텐츠 운영 품질을 표준화했다.

이력서용 축약:
- Python·Playwright 기반 네이버 블로그 프로모션 자동화 시스템 구축
- 엑셀 파싱, 브랜드별 HTML 디자인, 섹션별 PNG 캡처, 정리 엑셀 생성 자동화
- 네이버 SmartEditor 인용구·이미지·표 삽입 및 임시저장 자동화
- 브랜드 미등록 시 설정 자동 생성, DONE 상태 관리, 재실행 옵션 구현
- 블로그 본문 추출기와 카테크 콘텐츠 수집기를 함께 구축해 콘텐츠 운영 반복 업무 절감
