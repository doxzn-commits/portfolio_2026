# 네이버 인플루언서 추출기 프로젝트 기술 정리 및 포트폴리오 문장

## 1. 프로젝트 개요

네이버 인플루언서 추출기는 네이버 블로그 자동차 카테고리에서 협업 후보 인플루언서를 수집하고 Google Sheets에 정리하는 Python 자동화 도구다.

마케팅 협업 대상을 찾는 과정은 수작업으로 진행하면 페이지를 열고, 작성자명을 확인하고, 블로그 URL을 복사하고, 인플루언서 ID를 정리하는 반복 작업이 필요하다. 이 프로젝트는 Playwright로 네이버 페이지를 탐색하고, author 요소에서 후보 정보를 추출한 뒤, Google Sheets에 append하는 방식으로 후보 발굴 업무를 자동화했다.

---

## 2. 해결한 업무 문제

인플루언서 협업 후보를 수작업으로 찾을 때의 문제:
- 네이버 블로그 자동차 카테고리를 여러 페이지 반복 탐색해야 함
- 작성자명, 블로그 URL, 블로그 ID, 글 제목을 매번 수동 복사해야 함
- 수집 날짜와 후보 목록을 일관된 형식으로 정리하기 어려움
- 협업 대상 발굴과 실제 제안 업무 사이에 불필요한 정리 시간이 발생함

이 도구는 후보 수집과 시트 적재를 자동화해 인플루언서 발굴 업무의 초기 리서치 시간을 크게 줄이는 것을 목표로 한다.

---

## 3. 주요 기능

핵심 파일:
naver_influencer_scraper.py

주요 기능:
- Playwright로 네이버 블로그 자동차 카테고리 페이지 접근
- a.author 셀렉터 기반 작성자 요소 수집
- 작성자명 추출
- 블로그 URL 추출
- 블로그 URL에서 인플루언서 ID 추출
- 게시글 제목 추출
- 수집 날짜 자동 입력
- Google Sheets worksheet에 append
- 1~5페이지 반복 수집

대상 URL 패턴:
https://section.blog.naver.com/ThemePost.naver?directoryNo=25&activeDirectorySeq=3&currentPage={page}

Google Sheets 연동:
- gspread 사용
- 서비스 계정 인증 사용
- naver_influencer 워크시트에 결과 append

수집 데이터 형태:
- 수집일
- 이름
- 인플루언서 ID
- 블로그 URL
- 글 제목
- 출처 페이지

## 4. 실행 방식

실행 파일:
run_naver_influencer.bat

배치 파일은 작업 폴더로 이동한 뒤 Python 스크립트를 실행한다.

실행 명령:
python naver_influencer_scraper.py

비개발자도 더블클릭으로 실행할 수 있도록 배치 파일을 둔 점이 운영 편의성 측면에서 중요하다.

## 5. 협업 가이드 산출물

폴더에는 카피아 인플루언서 협업 가이드.pdf가 포함되어 있다.

이 문서는 수집된 인플루언서 후보에게 제안하거나 내부 협업 운영 기준을 설명하기 위한 자료로 볼 수 있다. 즉, 프로젝트는 단순 수집 스크립트만이 아니라 후보 발굴, 리스트화, 협업 제안 자료까지 이어지는 인플루언서 영업/제휴 운영 흐름의 일부다.

## 7. 업무적 성과

- 네이버 블로그 자동차 카테고리의 인플루언서 후보 탐색을 자동화했다.
- 작성자명, 블로그 URL, 인플루언서 ID, 글 제목을 일관된 형식으로 수집했다.
- Google Sheets append를 통해 후보 리스트 관리 흐름을 표준화했다.
- 배치 파일 실행 구조로 비개발자도 사용할 수 있는 형태로 만들었다.
- 협업 가이드 PDF와 함께 후보 발굴부터 제안 준비까지 이어지는 업무 흐름을 구성했다.
- 반복적인 후보 탐색과 리스트 정리 시간을 크게 절감하고, 협업 리드 발굴 속도를 높였다.

---

## 8. 포트폴리오용 문장

한 줄 소개:
네이버 블로그 자동차 카테고리에서 협업 후보 인플루언서를 자동 수집하고 Google Sheets에 정리하는 Python·Playwright 기반 리드 발굴 자동화 도구를 구축했다.

상세 문장:
인플루언서 제휴 마케팅을 위한 후보 발굴 업무를 자동화하기 위해 네이버 블로그 자동차 카테고리 수집기를 구현했다. Playwright로 네이버 카테고리 페이지를 순회하며 작성자명, 블로그 URL, 인플루언서 ID, 게시글 제목을 추출하고, gspread를 통해 Google Sheets에 자동 적재하도록 구성했다. 배치 파일 실행 구조를 제공해 비개발자도 사용할 수 있게 했으며, 협업 가이드 PDF와 함께 후보 발굴부터 제안 준비까지 이어지는 인플루언서 영업 프로세스를 효율화했다.

이력서용 축약:
- Python·Playwright 기반 네이버 인플루언서 후보 수집기 구현
- 네이버 자동차 카테고리 1~5페이지 작성자 정보 자동 추출
- 블로그 URL에서 인플루언서 ID 파싱 및 Google Sheets 자동 적재
- Windows batch 실행 파일 제공으로 운영자 사용성 개선
- 인플루언서 협업 후보 발굴 및 제안 준비 업무 시간 절감
