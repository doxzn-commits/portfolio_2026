# 뉴스 기사 생성 프로세스 히스토리 인계서

원본 공유 링크: https://claude.ai/share/83981a7b-549e-466d-af36-885223563bee

## 1. 주제와 목적
사용자가 Google News URL을 기반으로 카피아 뉴스 기사를 생성하려고 한 사례다.

제공된 URL은 Google News 리다이렉트 주소라 직접 접근이 어려웠다. Claude는 원문 URL 직접 fetch가 불가능하다고 안내하고, 당시 콘텐츠 가치가 높다고 판단한 자동차 뉴스 소재를 찾아 카피아 뉴스/프로모션형 콘텐츠로 재가공했다.

## 2. 처리된 소재
선택된 소재:
- 기아 EV9 3월 한정 재고 프로모션.
- 재고 차량 500만 원 즉시 할인.
- 서울 기준 보조금 합산 시 최대 1,000만 원 이상 혜택 가능.

콘텐츠 성격:
- 뉴스 콘텐츠.
- 프로모션 정보.
- 리스/상담 CTA로 자연 연결되는 카피아형 기사.

## 3. 적용된 기사 생성 기준
마스터 프롬프트 체크리스트로 언급된 기준:
- H1은 브랜드/이슈 + 독자 임팩트 공식.
- H2는 키워드 리스트형 3줄.
- H3는 골드 언더라인 + 좌측 바 스타일.
- 카테고리는 자동차 소식 단일.
- SEO 메타태그 4종 포함.
- Schema.org 구조화 데이터 포함.
- 스펙 비교표 2개 포함.
- CTA는 리스 상담으로 자연스럽게 연결.
- 과장 표현 금지.
- 확인된 수치만 사용.
- 이미지가 720px 미만이면 플레이스홀더 처리.

## 4. 산출물
대화상 HTML 퍼블리싱 파일과 JSON 출력 파일이 생성되었다. 실제 파일명은 공유 대화 본문에 노출되지 않았다.

## 5. 다음 AI 주의사항
- Google News 리다이렉트 URL은 직접 접근이 어려울 수 있다.
- 가능하면 원 기사 URL을 사용자에게 다시 요청한다.
- 원문에 접근하지 못한 경우 대체 소재 사용 사실을 명확히 알려야 한다.
- 뉴스 기사에는 확인된 수치만 사용한다.
- 과장된 혜택 표현을 피한다.
- 카피아 뉴스는 단순 보도보다 상담/리스 CTA와 연결되는 구조가 선호된다.
