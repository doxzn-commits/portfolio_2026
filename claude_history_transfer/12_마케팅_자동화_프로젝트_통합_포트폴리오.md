# 마케팅 자동화 프로젝트 통합 포트폴리오 정리

이 문서는 아래 4개 프로젝트를 포트폴리오 기재용으로 통합 정리한 문서다.

대상 프로젝트:
1. SEO/GEO 진단기
2. 블로그 자동화
3. 인플루언서 대시보드
4. 네이버 인플루언서 추출기

---

## 1. 전체 요약

카피아의 마케팅 운영 과정에서 반복적으로 발생하던 SEO 진단, 블로그 콘텐츠 제작, 인플루언서 정산, 상담 DB 분배, 협업 후보 발굴 업무를 자동화했다.

각 프로젝트는 개별 도구처럼 보이지만 공통 목표는 같다. 사람이 매월 반복하던 데이터 정리와 콘텐츠 운영 업무를 자동화하고, 결과를 대시보드·리포트·Google Sheets·네이버 블로그 임시저장 글처럼 바로 실행 가능한 형태로 전환하는 것이다.

사용한 기술은 TypeScript, Node.js, Express, Python, Playwright, Google Apps Script, Google Sheets, HTML, Vanilla JavaScript, Chart.js, SheetJS 등이다.

---

## 3. SEO/GEO 진단기

SEO/GEO 진단기는 웹사이트 URL을 입력하면 SEO 점수와 GEO 점수를 함께 산출하는 TypeScript 기반 분석 도구다. SEO는 기술적 SEO, 콘텐츠 품질, 온페이지 최적화, 백링크 권위, 사용자 경험으로 나누고, GEO는 인용 가능성, 권위·신뢰성, AI 친화 구조, 답변 일치도, 멀티소스 노출로 나누어 측정한다.

CLI와 Express 기반 웹 UI를 모두 지원하며, JSON/HTML/Markdown 개선 리포트를 생성한다. 타깃 질문을 입력하면 해당 질문과 콘텐츠 구조의 일치 여부도 분석한다.

포트폴리오 문장:
SEO와 GEO, 즉 AI 인용 최적화를 동시에 진단하는 TypeScript 기반 내부 분석 도구를 기획·구현했다. URL을 입력하면 기술 SEO, 콘텐츠 품질, 온페이지 최적화뿐 아니라 정의형 문장, FAQ 구조, 답변 일치도, AI 인용 가능성을 100점 체계로 분석하고, HTML/JSON/Markdown 리포트와 개선 우선순위를 자동 생성하도록 구성했다.

---

## 4. 블로그 자동화

블로그 자동화는 월별 수입차 프로모션 엑셀 파일을 기반으로 네이버 블로그 콘텐츠를 자동 생성하고 임시저장까지 진행하는 Python·Playwright 기반 시스템이다.

엑셀 파일에서 브랜드, 모델, 연식, 차량가격, 자사금융, 현금/타사 조건을 파싱하고, 브랜드별 HTML 디자인과 섹션별 PNG 이미지를 생성한다. 이후 Playwright로 네이버 SmartEditor를 조작해 인사말, 인용구, 이미지, 설명문, 실제 네이버 표, 해시태그를 자동 입력하고 임시저장한다.

포트폴리오 문장:
월별 수입차 프로모션 엑셀을 네이버 블로그 임시저장 글까지 자동 변환하는 Python·Playwright 기반 콘텐츠 운영 자동화 시스템을 구축했다. 엑셀 파싱, 브랜드별 HTML 디자인, 섹션별 PNG 캡처, 정리 엑셀 생성, 네이버 SmartEditor 입력을 하나의 흐름으로 연결해 매월 반복되던 콘텐츠 제작·정리·업로드 업무 시간을 크게 절감했다.

---

## 5. 인플루언서 대시보드

인플루언서 대시보드는 인플루언서별로 분산된 정산 시트와 상담 DB를 Google Apps Script 기반으로 통합 관리하는 사내 자동화 시스템이다.

syncInfluencers.gs는 인플루언서별 시트를 순회하며 월별 상담 DB, 전환율, 정산금액을 계산하고 마스터 시트에 반영한다. dbDistributor.gs는 원본 상담 DB에서 레퍼러 ID를 추출해 각 인플루언서 개별 시트에 자동 분배하고, 연락처 마스킹과 중복 방지를 수행한다. HTML 대시보드는 KPI, 월별 추이, 랭킹, 정산 대기 현황을 시각화한다.

포트폴리오 문장:
분산된 인플루언서 정산 시트와 상담 DB를 Google Apps Script 기반으로 자동 통합·분배하고, 단일 HTML 대시보드로 성과와 정산 상태를 확인하는 운영 시스템을 구축했다. 월별 상담 DB, 전환율, 정산금액 계산을 자동화하고 수기 입력 정산일자를 보존하도록 설계해 정산 누락 리스크와 월간 운영 리드타임을 크게 줄였다.

---

## 6. 네이버 인플루언서 추출기

네이버 인플루언서 추출기는 네이버 블로그 자동차 카테고리에서 협업 후보 인플루언서를 자동 수집해 Google Sheets에 정리하는 Python 자동화 도구다.

Playwright로 네이버 카테고리 페이지를 순회하며 작성자명, 블로그 URL, 인플루언서 ID, 게시글 제목을 추출하고, gspread로 시트에 append한다. 배치 파일을 함께 제공해 비개발자도 실행할 수 있도록 구성했다.

포트폴리오 문장:
네이버 블로그 자동차 카테고리에서 협업 후보 인플루언서를 자동 수집하고 Google Sheets에 정리하는 Python·Playwright 기반 리드 발굴 자동화 도구를 구축했다. 작성자명, 블로그 URL, 인플루언서 ID, 게시글 제목을 자동 추출해 후보 리스트화 업무를 줄이고, 협업 제안 준비 과정을 효율화했다.

## 7. 통합 포트폴리오 소개 문장

마케팅 운영 과정에서 반복적으로 발생하는 데이터 수집, 콘텐츠 생성, 성과 분석, 정산 관리 업무를 자동화하는 내부 도구들을 직접 기획하고 구현했다. TypeScript 기반 SEO/GEO 진단기, Python·Playwright 기반 네이버 블로그 자동화, Google Apps Script 기반 인플루언서 정산 대시보드와 DB 분배 시스템, 네이버 인플루언서 후보 수집기를 구축해 콘텐츠 운영과 마케팅 데이터 관리의 반복 업무 시간을 크게 절감했다.

- 반복적인 마케팅 운영 업무를 자동화하여 월간 콘텐츠 제작·정산·리서치 시간을 크게 절감

## 9. 면접 답변용 30초 소개

카피아에서 마케팅 운영 업무를 자동화하는 내부 도구들을 직접 기획하고 구현했습니다. SEO/GEO 진단기는 URL을 입력하면 검색 최적화와 AI 인용 가능성을 점수화하고 개선 리포트를 자동 생성합니다. 블로그 자동화는 월별 프로모션 엑셀을 HTML, 이미지, 정리 엑셀, 네이버 임시저장 글까지 변환합니다. 인플루언서 대시보드는 분산된 정산 시트와 상담 DB를 Google Apps Script로 통합·분배하고, HTML 대시보드로 성과를 시각화합니다. 추가로 네이버 인플루언서 후보 수집기를 만들어 협업 리드 발굴도 자동화했습니다. 전체적으로 반복적인 콘텐츠 운영, 정산, 리서치 업무 시간을 크게 줄인 프로젝트입니다.
